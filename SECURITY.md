# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in this package, please report it responsibly. **Do not open a public issue.**

Email **security@sinemacula.co.uk** with:

- A description of the vulnerability
- Steps to reproduce or a proof of concept
- The affected version(s)

You will receive an acknowledgement within 48 hours. We will work with you to understand the issue and coordinate a fix
before any public disclosure.

## Disclosure Policy

- Vulnerabilities will be patched in a timely manner and released as a new version
- Credit will be given to the reporter unless they prefer to remain anonymous
- Public disclosure will be coordinated with the reporter after a fix is available
